## Por que as preferências de locale precisam de uma store persistente

_I18nService.tUser()_ resolve o locale para um usuário por uma cadeia de prioridades: preferência do
usuário, depois preferência do servidor, depois o padrão configurado. Para que essa cadeia funcione
entre reinicializações do bot, algo precisa lembrar essas preferências. A store é esse elemento. É
a única interface responsável por persistir e recuperar as escolhas de locale de usuários e servidores.

A store é um componente separado e substituível, não uma estrutura de dados in-memory fixa, porque
preferências de locale têm os mesmos requisitos de persistência que qualquer outra configuração do
usuário. Um usuário que escolhe português hoje espera que sua preferência ainda esteja ativa na
próxima vez que o bot reiniciar.

## Como o I18nStoreRegistry funciona

_I18nStoreRegistry_ é uma classe estática que mantém a referência da store ativa. O lifecycle do i18n
sempre chama _I18nStoreRegistry.get()_ na inicialização para recuperar a store, independentemente de
uma customizada ter sido registrada ou não. Se nenhuma store customizada foi definida, o registro cria
e retorna uma instância de _MemoryStore_ automaticamente. Nenhum erro é lançado e nenhuma configuração
extra é necessária para o uso básico.

A propriedade principal desse design é que o registro da store é independente do objeto de configuração
do plugin. Você chama _I18nStoreRegistry.set_ em qualquer momento antes do lifecycle do i18n ser
executado, seja no arquivo principal de bootstrap, dentro de um lifecycle hook customizado, ou em
qualquer outro ponto da sequência de inicialização do bot.

## Registrando uma store customizada

```typescript filename="src/main.ts"
import { I18nStoreRegistry } from '@spraxium/i18n';
import { SpraxiumFactory } from '@spraxium/core';
import { IntentPreset } from '@spraxium/common';
import { DrizzleLocaleStore } from './stores/drizzle-locale.store';
import { db } from './database/connection';

async function main(): Promise<void> {
  I18nStoreRegistry.set(new DrizzleLocaleStore(db));
  const app = await SpraxiumFactory.create({
    token: process.env.DISCORD_TOKEN!,
  });
  app.intents(IntentPreset.Standard);
  await app.listen();
}

main();
```

A store é definida antes de _SpraxiumFactory.create_ ser chamado, garantindo que esteja disponível
antes do lifecycle do i18n ser executado. Registrá-la no arquivo de bootstrap principal mantém a
dependência explícita e visível.

## Implementando uma store customizada

Qualquer classe que implementa _I18nStore_ é uma store válida. Use a ORM que já faz parte do projeto:

```typescript filename="stores/drizzle-locale.store.ts"
import { eq } from 'drizzle-orm';
import type { I18nStore } from '@spraxium/i18n';

export class DrizzleLocaleStore implements I18nStore {
  constructor(private readonly db: DrizzleDatabase) {}

  async init(): Promise<void> {
    await this.db.execute(sql`       CREATE TABLE IF NOT EXISTS locale_preferences (
        user_id TEXT PRIMARY KEY, locale  TEXT NOT NULL
      )
    `);
  }

  async getUser(userId: string): Promise<string | undefined> {
    const linha = await this.db
      .select()
      .from(localePreferences)
      .where(eq(localePreferences.userId, userId))
      .limit(1);
    return linha[0]?.locale ?? undefined;
  }

  async setUser(userId: string, locale: string): Promise<void> {
    await this.db
      .insert(localePreferences)
      .values({ userId, locale })
      .onConflictDoUpdate({ target: localePreferences.userId, set: { locale } });
  }

  async getGuild(guildId: string): Promise<string | undefined> {
    const linha = await this.db
      .select()
      .from(guildLocalePreferences)
      .where(eq(guildLocalePreferences.guildId, guildId))
      .limit(1);
    return linha[0]?.locale ?? undefined;
  }

  async setGuild(guildId: string, locale: string): Promise<void> {
    await this.db
      .insert(guildLocalePreferences)
      .values({ guildId, locale })
      .onConflictDoUpdate({ target: guildLocalePreferences.guildId, set: { locale } });
  }
}
````

```typescript filename="stores/prisma-locale.store.ts"
import type { I18nStore } from '@spraxium/i18n';
import type { PrismaClient } from '@prisma/client';

export class PrismaLocaleStore implements I18nStore {
  constructor(private readonly prisma: PrismaClient) {}

  async init(): Promise<void> {}

  async getUser(userId: string): Promise<string | undefined> {
    const registro = await this.prisma.localePreference.findUnique({
      where:  { userId },
      select: { locale: true },
    });
    return registro?.locale ?? undefined;
  }

  async setUser(userId: string, locale: string): Promise<void> {
    await this.prisma.localePreference.upsert({
      where:  { userId },
      create: { userId, locale },
      update: { locale },
    });
  }

  async getGuild(guildId: string): Promise<string | undefined> {
    const registro = await this.prisma.guildLocalePreference.findUnique({
      where:  { guildId },
      select: { locale: true },
    });
    return registro?.locale ?? undefined;
  }

  async setGuild(guildId: string, locale: string): Promise<void> {
    await this.prisma.guildLocalePreference.upsert({
      where:  { guildId },
      create: { guildId, locale },
      update: { locale },
    });
  }
}
````

```typescript filename="stores/typeorm-locale.store.ts"
import { Entity, Column, PrimaryColumn, DataSource, Repository } from 'typeorm';
import type { I18nStore } from '@spraxium/i18n';

@Entity('locale_preferences')
class LocalePreference {
  @PrimaryColumn()
  userId!: string;
  @Column()
  locale!: string;
}

@Entity('guild_locale_preferences')
class GuildLocalePreference {
  @PrimaryColumn()
  guildId!: string;
  @Column()
  locale!: string;
}

export class TypeOrmLocaleStore implements I18nStore {
  private usuarios: Repository<LocalePreference>;
  private servidores: Repository<GuildLocalePreference>;

  constructor(private readonly dataSource: DataSource) {
    this.usuarios = dataSource.getRepository(LocalePreference);
    this.servidores = dataSource.getRepository(GuildLocalePreference);
  }

  async init(): Promise<void> {
    await this.dataSource.synchronize();
  }

  async getUser(userId: string): Promise<string | undefined> {
    const registro = await this.usuarios.findOne({ where: { userId } });
    return registro?.locale ?? undefined;
  }

  async setUser(userId: string, locale: string): Promise<void> {
    await this.usuarios.upsert({ userId, locale }, ['userId']);
  }

  async getGuild(guildId: string): Promise<string | undefined> {
    const registro = await this.servidores.findOne({ where: { guildId } });
    return registro?.locale ?? undefined;
  }

  async setGuild(guildId: string, locale: string): Promise<void> {
    await this.servidores.upsert({ guildId, locale }, ['guildId']);
  }
}
````

## Um comando de mudança de locale que escreve na store

```typescript filename="locale.handler.ts"
import { Ctx, SlashCommandHandler } from '@spraxium/common';
import { I18nService } from '@spraxium/i18n';
import type { ChatInputCommandInteraction } from 'discord.js';
import { LocaleCommand } from '../commands/locale.command';

@SlashCommandHandler(LocaleCommand)
export class LocaleHandler {
  constructor(private readonly i18n: I18nService) {}

  async handle(@Ctx() interaction: ChatInputCommandInteraction): Promise<void> {
    const locale = interaction.options.getString('locale', true);
    const userId = interaction.user.id;

    await this.i18n.setUserLocale(userId, locale);

    const confirmacao = await this.i18n.tUser(
      userId,
      'commands.locale.alterado',
      { locale },
    );

    await interaction.reply({ content: confirmacao, ephemeral: true });
  }
}
````

## Reiniciando a store em testes

A natureza estática do _I18nStoreRegistry_ significa que o estado pode vazar entre casos de teste.
Use _reset()_ nos hooks de teardown para manter os testes isolados:

<CodeTabs>
  <Tab label="Vitest" language="typescript">
```typescript filename="test/i18n.test.ts"
import { I18nStoreRegistry, MemoryStore } from '@spraxium/i18n';

beforeEach(() => {
  I18nStoreRegistry.set(new MemoryStore());
});

afterEach(() => {
  I18nStoreRegistry.reset();
});

````
  </Tab>
  <Tab label="Store com spy" language="typescript">
```typescript filename="test/locale.handler.test.ts"
import { I18nStoreRegistry } from '@spraxium/i18n';
````
